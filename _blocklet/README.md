# 链网超级搜索框

## 简介

> 查询官方链网的几条链是否存在对应的记录：账户、交易、资产
> 展示命中记录类型、属于哪条链、命中记录的摘要卡片

## 安装项目依赖

```
yarn install
```

### 启动开发环境

#### 前端

```
yarn serve
```

#### 后端

```
node server/app.js
```

### 编译生产内容

```
yarn build
```

### 部署 abtnode 上运行

```
abtnode deploy .
```
